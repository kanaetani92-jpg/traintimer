# Stage 31：スマホ縦画面の最適化 検証メモ

## 実装範囲

- iPhone / Androidスマホの縦画面を1カラム順に再調整。
- タイマー画面の表示順を、赤いヘッダー → 線路切替 → 線路表示エリア → 現在区間・次の駅 → 操作ボタン → ＋1分・＋5分 → 下部ナビで維持。
- タイマー画面に運行中の電車リストは表示せず、「運行中の電車：○本」の小表示だけを維持。
- 固定下部ナビと操作ボタンが重ならないよう、100dvh・safe-area・Grid/Flexを使って再調整。
- 359px以下の狭い画面では、操作ラベルを省略し、余白・文字サイズ・線路高を圧縮。
- 横線路・縦線路・円形線路の3形状でスマホ縦表示を確認。
