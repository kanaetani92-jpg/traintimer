# スマホ版画面設計：経路案内型

## 1. デザイン方針

- 白を中心に、現在地・主操作だけ赤で強調する
- 時刻と次の駅を最優先で表示する
- 経路は路線色と縦線で上から下へ追えるようにする
- 角丸や影は控えめにし、情報の区切りを枠線で示す
- 子ども向けの分かりやすさを残しながら、公共交通アプリのように整理する
- 特定サービスのロゴや画面を複製せず、情報設計のみを参考にする

## 電車の表示

横線路・縦線路では、電車の横や下に進行方向の矢印を表示しません。線路の配置と電車の移動そのもので進行方向を示します。

## 文字設定

日本語フォントは、端末にインストールされている場合に次の順で使用します。

1. BIZ UDPGothic / BIZ UDゴシック
2. Meiryo / メイリオ
3. 端末標準のsans-serif

外部Webフォントは使用せず、オフライン動作を維持します。

## 推奨ページ構成

スマホ版は「タイマー・経路編集・Myタイマー・設定」の4ページ構成とします。詳細は `page-structure-design.md` を参照してください。タイマーページではスタートからゴールまでの線路を一画面内に収めます。



## 実装メモ

- 現在は4ページ構成（タイマー / 経路編集 / My設定 / 設定）です。
- タイマー画面には、スタートからゴールまでの線路と、残り時間・操作ボタンを同じページに配置します。
- 円形線路中央の残り時間は、全体 / 区間 / 両方 / 非表示を切り替えられます。


## 案1：上タイマー・中央線路・下操作

- 上部にゴールまでの時間と、この区間の時間を表示します。
- 中央にスタートからゴールまでの線路を表示します。
- 線路の下に現在区間と次の停車駅を配置します。
- 最下部に開始または一時停止、＋1分、＋5分を配置します。
- リセットはハンバーガーメニューへ移します。
- 通常時はスマートフォン縦置きでスクロールせず確認できます。

## 一時停止時の操作

一時停止すると、主操作部分を次の2ボタンへ切り替えます。

```text
┌──────────────┐ ┌──────────────┐
│ ▶ つづきから  │ │ ↺ はじめから  │
└──────────────┘ └──────────────┘
```

- 「つづきから」：停止した残り時間と電車位置から再開します。
- 「はじめから」：設定時間へ戻し、スタート地点からすぐに再開します。
- 開始前は「はじめる」だけ、運転中は「一時停止」だけを表示します。
