# でんしゃタイマー 静かモード・線路色切替版

円形の線路を電車が走る、子ども向けのWebタイマーです。
この版では、スタート前から分割された線路が見える状態を保ちつつ、保護者設定で「単色の線路」と「色違いの線路」を選べるようにしました。
また、初期状態は「動きを少なくする（静かモード）」をオンにしています。

## 主な機能

- スタート前から見える円形線路の分割ガイド
- 線路の色を「単色」「色違い」から選択
- 初期設定で静かモードをオン
- 分割ごとの線路の切れ目表示
- 切れ目に小さな丸い目印を表示
- 動く電車アイコン
- 次に向かう駅の発光表示
- 到着済み駅の色変更
- ゴール時の旗表示
- 時間プリセットボタン
- 駅数プリセットボタン
- しゅっぱつ、とまる、またすすむ、はじめにもどる
- 保護者設定ボトムシート
- localStorageによる設定保存
- Wake Lock、振動、ゴール音の任意設定
- PWA用 manifest / service worker

## 使い方

1. `index.html` をブラウザで開きます。
2. 時間を入力、またはプリセットから選びます。
3. 駅の数を入力、またはプリセットから選びます。
4. 保護者設定から、線路の色を「単色」または「色違い」に変更できます。
5. 「しゅっぱつ」を押すと、電車が分割された線路を進みます。

## 今回のUI/UX修正

- 「動きを少なくする（静かモード）」を初期状態でオンにしました。
- 保護者設定に「線路の色」を追加しました。
- 線路の色を「単色」と「色違い」から選択できるようにしました。
- 色変更は保存前でも画面上で確認でき、保存すると次回以降も反映されます。
- 単色表示では、色の多さが気になる子どもにも使いやすいようにしました。
- 色違い表示では、分割された線路の区間が見た目で分かりやすい状態を保ちました。

## 注意

Wake Lock、振動、音は端末やブラウザの対応状況により動作が異なります。
子どもが音、振動、動き、色の多さに敏感な場合は、保護者設定で調整してください。
