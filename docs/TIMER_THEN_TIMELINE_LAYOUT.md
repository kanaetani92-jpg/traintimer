# タイマー操作とタイムラインの表示順修正

## 目的

PC版で、タイマー操作の下にタイムラインが来るように表示順を整理しました。

## 修正内容

### CSSで順番を明示

次の順番になるようにしました。

```text
1. 大きい絵カード
2. できた！／前の駅にもどる
3. タイマー操作
4. タイムライン
5. ゴール表示
```

### PC版

PC版では、`#viewTimerControls` の後に `.track-wrap` が表示されるように `order` を指定しました。

### スマホ版

スマホ版も同じ考え方で、タイマー操作の下にタイムラインが来るようにしました。
