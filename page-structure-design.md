# 4ページ構成・案2レイアウト

## タイマー画面

線路を画面の主役にし、残り時間を線路表示の中へ統合します。

- 横・縦・円の切り替え
- スタートからゴールまでの全線路
- 線路内の残り時間
- 乗車中の区間
- 次の停車駅
- 開始・一時停止
- ＋1分・＋5分

独立した大きなタイマーカードは画面上に表示しません。

## 残り時間表示

設定から次の4種類を選べます。

1. 全体の残り時間
2. その区間の残り時間
3. 全体とその区間の両方
4. 表示しない

横線路では線路上部、縦線路では線路右側、円形線路では円の中央に表示します。

## タイマー操作の状態別表示

| 状態 | 表示する主操作 |
|---|---|
| 開始前 | はじめる |
| 運転中 | 一時停止 |
| 一時停止中 | つづきから / はじめから |
| 終了 | もういちど始める |

## 経路編集と設定の役割分離（1.7.0）

- 経路編集ページでは、駅・区間・単位・線路形状を直接編集します。
- 「経路をくわしく編集する」と「条件を変更」は削除しました。
- 右上の歯車は、表示・音・アクセシビリティ設定だけを開きます。
- 経路編集とアプリ全体の設定は、同じ画面へ移動しません。
