# 第4段階：タブレット横・PCの2カラム最適化 検証メモ

## 実装範囲

- 対象：タブレット横、900px以上1199px以下の小型PC、1200px以上のPC
- タイマー画面を「左：線路・電車・残り時間」「右：現在区間・次の駅・操作ボタン・＋1分・＋5分」に再配置
- My設定ページを「すぐ使う」「運行中の電車」「登録した経路」「データ管理」のカード構成に整理
- 経路編集ページを「左：現在の設定」「右：駅・区間・線路形状の編集」に整理
- 設定ページを「音・表示」と「操作・安全・画面維持」が読みやすい2カラム寄りの配置に整理

## 主な変更点

### HTML

- `journey-status-grid` と `activeTrainSummary` を `routeSection` から `timerActionsSection` へ移動
  - スマホでは線路の下、タブレット横・PCでは右側操作パネルに自然に表示されるようにした
- `data-management` を `preset-card` の内部から `pagePresets` 直下の独立カードへ移動
  - PCで「すぐ使う」「運行中の電車」「登録した経路」「データ管理」を明確に分けられるようにした

### CSS

- 900px以上で `#pageTimer` を2カラム化
- 1200px以上でPC向けの幅と余白を拡張
- タイマー操作パネルを右側にまとめ、`position: sticky` で見つけやすくした
- My設定ページを900〜1199pxでは2列、1200px以上では3列＋データ管理下部に整理
- 経路編集ページを900px以上で左右2カラム化
- 設定ページの設定一覧を2列にし、音・表示とロック・画面維持が読みやすい配置にした
- `[hidden]` がCSSの `display:grid` 指定に上書きされないよう、非表示安全ルールを追加
